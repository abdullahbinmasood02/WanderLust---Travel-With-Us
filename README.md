# Wanderlust — Premium Travel Booking 

Welcome to **Wanderlust**, a premium front-end travel booking application built to provide a visually stunning and seamless user experience for exploring and booking world-class travel destinations.

## Features

- **Interactive Destination Discovery**: Search, filter, and sort handpicked journeys across 6 continents. Find the perfect trip by price, rating, or duration.
- **Dynamic Booking System**: Select trips, input traveler details, and get instant, dynamically calculated pricing summaries.
- **Cart Management**: Easily keep track of all your selected trips with a built-in sidebar cart.
- **Advanced Insights Panel**: View dynamic data breakdowns, including trips by continent, cheapest available options, and seasonal data.
- **Real-Time Dashboards**: Features a live countdown to upcoming adventures and a comprehensive statistics panel tracking trip metrics.

## Technologies Used

This project is built using core web technologies with zero external dependencies:
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom modern styling, including flexbox, CSS grid, variables, and responsive design techniques.
- **JavaScript (ES6+)**: Powers the core logic, dynamic state management, advanced DOM manipulation, and data handling.


## Project Structure

- `index.html` - The main structure and layout of the application.
- `style.css` - All styling, theme definitions, and animations.
- `script.js` - The core JavaScript logic that drives interactivity and data management.
